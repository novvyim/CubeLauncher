## 📖 About

**CubeLauncher** is a cross-platform desktop application built to simplify the process of creating and managing Minecraft servers.

The goal of the project is to eliminate repetitive tasks such as:
- manually downloading server cores,
- editing configuration files by hand,
- maintaining multiple isolated test environments.

Whether you're a developer, tester, or just a Minecraft enthusiast — CubeLauncher helps you focus on what matters: your gameplay and your code.

<p align="center">
  <img src="public/dashboard.png" width="850" alt="CubeLauncher Dashboard">
</p>

## ✨ Features

- ✅ One-click installation for **Vanilla**, **Paper**, and **Fabric** servers
- 🗂️ Multiple isolated server instances
- 📁 Built-in file manager
- 📊 Real-time server monitoring (**CPU**, **RAM**, **TPS**)
- ⚙️ Edit `.properties` and `.yml` configs directly in-app
- 🎛️ Simple start/stop controls
