Contributing to CubeLauncher
Thank you for your interest in contributing to CubeLauncher! We welcome all contributions, including code improvements, bug reports, and documentation updates.
How to contribute:
Fork the repository to your own GitHub account.
Create a new branch for your feature or bug fix (git checkout -b feature/my-feature).
Commit your changes with a clear and descriptive message.
Push to the branch and create a Pull Request.
Wait for review: I will check your code as soon as possible.
Reporting Issues:
If you find a bug or have a suggestion, please open an Issue and describe it clearly.
Code Style:
Please try to keep your code clean and follow the existing style of the project.
